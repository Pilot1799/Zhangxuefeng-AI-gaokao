## 🛑 新手小白必看：完整的软件去哪下载？！

⚠️ **千万不要在左边这堆英文代码文件里乱点！** ⚠️

如果你不懂怎么编译代码，只想在自己电脑上跑起这套系统，请严格按照以下步骤去下载**包含绝密高考数据库的最终打包压缩包（大约500多MB）**：
<img width="2508" height="1279" alt="1513f84f7d7d334fedfacc91fe36d77b" src="https://github.com/user-attachments/assets/feef94de-8405-45a8-b483-8b57484cc691" />
<img width="2462" height="1279" alt="fb7e5d09bf71e19e506221afa407794a" src="https://github.com/user-attachments/assets/d4c5097a-e8ec-43a4-9273-3cd88c217de7" />

1. 👉 看向你电脑屏幕的**最右侧边栏**。
2. 往下滑一点点，找到 **【发布】(Releases)** 区域。
3. 点击那个带有绿色 **[最新版 / Latest]** 标签的链接（例如：*张雪峰 ai 高考志愿系统首发版*）。
4. 页面跳转后，一直滑到屏幕最底端，找到 **【Assets】(资源)**。
5. **下载体积最大（几百兆）的那个 `.zip` 压缩包即可！** 
> 💡 *小提示：千万别下错成名字叫 `Source code` 的文件，那个几十KB的空壳子电脑跑不起来！*

## 🛠️ 下载完解压后怎么用？（专治手残党）

你根本不需要会写任何代码！把那个几百兆的压缩包解压后：

1. **傻瓜式配置环境**：找到文件夹里的 `00_一键安装初始环境.bat`，双击它！黑框框会自动帮你解决环境配置的问题，不需要你懂 Node.js 和向量数据库。
2. **看一眼使用说明**：为了让你顺利使用，我在文件夹里为你准备了一份 `小白使用说明.docx` 的 Word 文档，请一定要看！尤其是里面**教你如何向 AI 提问**的必胜句式！
3. **日常启动系统**：以后每次使用，只要双击 `一键启动_局域网前后端.bat` 就能在浏览器开启咨询啦。

*遇到看不懂的报错？别内耗，直接复制报错连同我的说明文档发给 Kimi、豆包 或 ChatGPT，它们会一字一句教你怎么解决。*
